# Demo Project

This is a demo project with the following components:
- **Python Script:** `main.py`
- **HTML File:** `index.html`
- **CSS File:** `styles.css`
- **JavaScript File:** `script.js`

## How to Run
1. Run the Python script:
    ```bash
    python3 main.py
    ```

2. Open the `index.html` file in your web browser.
